# Job Application Tracker

A simple full-stack job application tracker that helps you organize and monitor your job hunt in one place.
Track companies, roles, application status, and notes without relying on messy spreadsheets.

This project is split into a frontend client and a backend server, making it easy to develop and scale independently.

## Why this project exists

Job hunting usually means juggling multiple applications across different platforms. This app was built to:
-  Keep all applications in one place
-  Track progress and application stages clearly
-  Make filtering and reviewing applications quick and painless
-  Serve as a practical full-stack JavaScript project

## Tech stack

### Frontend
- React
- Tailwind

### Backend
- Node.js
- Express
- RESTful API design

### Database
- PostgreSQL

## Core features

- Authentication
- Create, edit, and delete job applications
- Track application status (Applied, Interviewing, Offer, Rejected, etc.)
- Store company name, role, date applied, job url, salary range, and notes
- Filter and search applications
